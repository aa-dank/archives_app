Archive Search
Search overview

Enter one or more search terms. Multiple plain terms are searched together, so annual report looks for both words. Use quotes for a phrase, such as "annual report".

Search terms support simple web-style operators: OR broadens a search, as in budget OR invoice, and a leading minus excludes a term, as in budget -draft. Parentheses, wildcards, and regular expressions are not supported.

Choose one search mode and one scope. Project and CAAN scopes use recorded project file-server locations; location scope accepts a copied Records path or a Records-relative folder prefix.

All-archive document text searches can take longer than scoped searches.

File Extension accepts one extension only, such as pdf or .pdf. Leave it blank to search all extensions; use the Excel workbook on the results page for additional extension filtering.
